## サービス概要
どんなサービスなのかを３行で説明してください。
同じコミュニティ内で気になるけど喋ったことのない人同士の、話すきっかけを作ります。
3メッセージ送ると一つのアイテム（ニット系のアイテムまたはキャラクターなど）になります。
アイテムは集めることもでき、人に送ることもできます。

## このサービスへの思い・作りたい理由
このサービスの題材となるものに関してのエピソードがあれば詳しく教えてください。
このサービスを思いつくにあたって元となる思いがあれば詳しく教えてください。

RUNTEQコミュニティをはじめ、どこのコミュニティでも話すきっかけがあれば仲良くなれるのにその一歩が踏み出せずに友達を増やしたいのに増やせない人々をたくさん見てきました。人は意外と思っているより優しいし、話しかけられるのを待っている。共通の趣味や好きなことがあれば尚更、きっかけさえあればすぐにお互い打ち解けられるはずと思います。
「きっかけ」さえあれば人は仲良くなれる。「きっかけ」さえあれば。
私は昔から知らない人に話しかけてすぐに仲良くなれるタイプなため、それができずに交友関係を広げられなくて困っている人を見る度に歯痒く思っていました。
思い返せば、RUNTEQに入る前に芸大に通い作品を作っていた頃から、人と人との関係性を編む（生む）きっかけ作りをテーマにニットの作品制作を行なっていました。作品とパフォーマンスを通して人と人が交流する（編む）きっかけ作りをしていました。
自分の制作のテーマをweb制作に置き換えたアイディアです。これならどんなに時間がかかっても、どんなにエラーに苦しめられてもこれだったら、途中に投げ出さずに作れると確信しました。
このサービスによって、コミュニティの人々が今まで以上にコミュニティを楽しめるようになって欲しいと強く思っています。

## ユーザー層について
決めたユーザー層についてどうしてその層を対象にしたのかそれぞれ理由を教えてください。
ユーザー層は、まずはRUNTEQコミュニティ。
20代から30代半ばくらい。

理由は、今現在私が所属していて、もっと期を超えた交流が受講生どうしてもっと盛んに起こったらいいのになぁと思っているからです。同期とは仲がいいだけど先輩や後輩に自分から話しかけられないから、きっかけを待っている人は多いと思います。私の同期も、私がXで先輩と絡んだ後にその先輩が私を目的に自分たちの仮住まいに来てくれることを待っていると話していました。
話しかけたいけどかけられない状態を打破できるサービスが生まれれば、そのコミュニティ内にいる人々ははもっと活発に有意義に楽しく充実した時間を過ごせると思います。

まだ先の話になりますが、ゆくゆくは、RUNTEQ以外のコミュニティにも広がって行けば嬉しいですが、まずはターゲットを絞りRUNTEQコミュニティとします。

## サービスの利用イメージ
ユーザーがこのサービスをどのように利用できて、それによってどんな価値を得られるかを簡単に説明してください。

- ユーザーはサイトにアクセス

*ログインした場合*

ログインしたらプロフィールに自己紹介文を書きます。
Xアカウントユーザー名も書く。
書いた自己紹介を元にタグがいくつかつくられます。
自分でタグを作る、または自己紹介文からAIで自動でタグを作る。
ユーザーは、タグ検索を元に質問したい人を探します。元々質問したい人がいる場合はタグ検索省略しても大丈夫。
名前と紹介文とタグのボードが掲示板一覧で探すこともできる。

紹介文の例：初めまして。〇〇です。音楽と映画が好きです。毎年フジロックに行ってます。色々好きですが特にくるりが好きです。映画もよく観ます。特に、A24の出している映画が好きで、アリ・アスター監督のホラーは独特で解釈のしがいがあるので好きです。皆さんとおしゃべりできると嬉しいです。

タグの例：音楽、フジロック、くるり、映画、アリ・アスター、ホラー、A24
AIを導入することで、より具体的なタグ生成がされそう。



*ログインしない場合*

プロフィールに自己紹介文は書けない。
タグ検索や検索機能を使って、話しかけたい人を探す。
名前と紹介文とタグのボードが掲示板一覧で見れる。
タグと検索機能、掲示板一覧で話しかけたい人を探すことができる。
ログインしない場合はここまでしかできない。
プロフィールに自己紹介文もかけないし、質問も送れないアイテムを獲得することもできない。


*ログインした場合のみの機能*
- 質問してアイテムをゲット
相手の方のプロフィールにある、「Xでメッセージを送る」ボタンをクリックする。
連絡フォームでメッセージを送るとその内容が相手のXのリプとなって送られます。
メッセージは動的OGPでイメージ画像にメッセージが載ったものを相手のリプに送信。
連絡フォームでメッセージを送るときに送信ボタンを押すと、アイテムが取得できます。

- 3人に質問すると、一つのアイテムをゲット。
アイテムは、編み物（コミュニティを編むをテーマにしてるので、セーターや帽子やズボンなど、、、）
このアイテムは獲得したら自身のプロフィールに保存されます。自分でコレクションにもでき、人にプレゼントすることもできます。
アイテムは、RUNTEQのバッチやカリキュラムの一単元が終わるごとに出てくるイラストのようにXに投稿できます。

## サービスの差別化ポイント・推しポイント
似たようなサービスが存在する場合、そのサービスとの明確な差別化ポイントとその差別化ポイントのどこが優れているのか教えてください。
独自性の強いサービスの場合、このサービスの推しとなるポイントを教えてください。

コミュニティを編むをテーマに、獲得アイテムをニットアイテムにする。
すでに認知はしている人に話しかけるきっかけを作るということを目的に置いている点。
自己紹介とタグ機能を使って、共通の話題を見つけやすくしている点。
話しかけることによって、アイテムがゲットできる点。
加えて、3メッセージを送らないとアイテムはゲットできないので、ゲーム感覚で強制的に3人または3回メッセージを送りたくなる。
取得したアイテムを交換できることによって、そこでまたコミュニケーションが生まれるし、仲良くなりたいアピールができる。
